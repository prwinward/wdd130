Overview

Purpose

The purpose of this website is to show how awesome rafting is. Rafting can be fun for anyone as long as they are safe while doing so.

Audience

My audience for this website is for all the people out there that are looking for fun while rafting. Rafting is for all those who know how to be safe and love to do it.
